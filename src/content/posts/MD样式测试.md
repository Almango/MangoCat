Markdown 全格式示例文档
一、标题格式（6 级）
Markdown 支持 6 级标题，通过 # 数量区分，# 后需加空格。
1 级标题（1 个 #）
2 级标题（2 个 #）
3 级标题（3 个 #）
4 级标题（4 个 #）
5 级标题（5 个 #）
6 级标题（6 个 #）

二、文本基础格式
1. 强调格式
斜体：*斜体文本* 或 _斜体文本_ → 斜体文本
粗体：**粗体文本** 或 __粗体文本__ → 粗体文本
粗斜体：***粗斜体文本*** 或 ___粗斜体文本___ → 粗斜体文本
删除线：~~删除线文本~~ → 删除线文本
下划线（HTML 兼容）：<u>下划线文本</u> → 下划线文本 - 高亮（部分编辑器支持）：==高亮文本== → == 高亮文本 ==
2. 引用格式
通过 > 表示引用，支持多级嵌套和混合格式。
一级引用：> 引用文本二级引用：>> 嵌套引用三级引用：>>> 多层嵌套

引用中可混合格式：粗体、斜体、链接

3. 换行与分隔线
换行：行尾加 2 个空格 + 回车，或直接空一行（推荐后者）
分隔线：---（三个及以上减号）、***（星号）、___（下划线），单独占一行，效果如下：




三、列表格式
1. 无序列表
支持 *、+、- 三种标记，标记后加空格，可嵌套。
无序列表项 1（- 文本）
嵌套无序列表 1.1
嵌套二级列表 1.1.1
无序列表项 2（+ 文本）
无序列表项 3（* 文本）
无序列表项 4
2. 有序列表
通过数字 + . + 空格表示，序号不影响渲染结果，支持嵌套。
有序列表项 1（1. 文本）
嵌套有序列表 1.1
嵌套有序列表 1.2
有序列表项 2（3. 文本，序号写 3 但渲染为 2）
有序列表项 3（5. 文本，序号写 5 但渲染为 3）
3. 任务列表（待办清单）
通过 - [ ]（未完成）和 - [x]（已完成）表示，[ ] 中需加空格。
已完成任务 1
已完成任务 2
未完成任务 1
未完成任务 2

2. 图片链接
格式：![图片alt文本](图片地址 "可选标题")，alt 文本为图片加载失败时显示的内容。

先定义链接标识，再引用，适合文档内多次使用同一链接。

五、代码格式
1. 行内代码
用 `（反引号）包裹，适合短代码或关键词。
示例：print("Hello Markdown") 是 Python 打印语句，HTML 中的 ` 标签。
2. 代码块
用 ```````` ``（三个反引号）包裹，支持指定语言高亮，不指定则默认无高亮。
（1）无语言指定
// 无语言高亮的代码块
function hello() {
  console.log("Hello World");
}

（2）指定语言（Python）
# Python 代码高亮示例
def add(a, b):
    return a + b

result = add(10, 20)
print(f"结果：{result}")  # 输出 30

七、脚注与注释
1. 脚注
格式：文本[^脚注标识] + 文档末尾定义 [^脚注标识]: 脚注内容
这是需要解释的文本 [^1]，还有另一个注释 [^note]。
2. 注释（HTML 兼容）
Markdown 原生无注释语法，可使用 HTML 注释 ``，注释内容不会渲染。
一条注释，在页面上不会显示 -->

八、数学公式（需 LaTeX 支持）
部分编辑器（如 Typora、GitBook）支持 LaTeX 数学公式，通过 $ 包裹。
1. 行内公式
格式：$公式内容$
示例：勾股定理 \(a^2 + b^2 = c^2\)，圆的面积 \(S = \pi r^2\)。
2. 块级公式
格式：$$公式内容$$（单独占一行）
\(\sum_{i=1}^n i = \frac{n(n+1)}{2} \\
\int_0^1 x^2 dx = \frac{1}{3}\)

九、特殊格式
1. 转义字符
使用 \ 转义 Markdown 特殊字符（#、*、[、] 等），使其显示原字符。
示例：
显示 # 文本：# 文本
显示 *斜体*：* 斜体 *
显示 [链接]()：\(é¾æ¥\)

2. 表情符号（部分编辑器支持）
使用 :emoji名称: 插入表情，不同平台支持的表情可能不同。
示例：😊 😂 👍 👎 🎉 📚 :heart: :star:
3. 目录（自动生成，部分编辑器支持）
使用 [TOC] 自动生成文档目录（需编辑器支持，如 Typora）：
[TOC]

定义（放在文档末尾） -->
[^1]: 这是脚注的具体内容，用于补充说明文本。
[^note]: 另一个脚注示例，可自定义标识（字母、数字均可）。
